Settings File
 Previous Top Next
All of the Figlut Web Service settings can be edited in the directly in the XML based settings file: FiglutServerSettings.xml. This file can be found directly in the Figlut Web Service's installation directory e.g. C:\Program Files\Figlut\Figlut Web Service\FiglutServerSettings.xml.

The file (with default settings) looks as follows:

<?xml version="1.0" encoding="utf-8"?>
<FiglutServerSettings xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns:xsd="http://www.w3.org/2001/XMLSchema">
  <ShowMessageBoxOnException>false</ShowMessageBoxOnException>
  <LogToFile>true</LogToFile>
  <LogToWindowsEventLog>true</LogToWindowsEventLog>
  <LogToConsole>true</LogToConsole>
  <LogFileName>Figlut.Server.Log.txt</LogFileName>
  <EventSourceName>Figlut.Server.Source</EventSourceName>
  <EventLogName>Figlut.Server.Log</EventLogName>
  <LoggingLevel>Maximum</LoggingLevel><!--None/Minimum/Normal/Maximum-->
  <DBMS>MsSqlServer</DBMS><!--MS SQL Server is currently the only supported DBMS.-->
  <DatabaseConnectionString>Data Source=PAULKOLOZSV4DD3\SQL2008STANDARD;Initial Catalog=DigisticsStoreDelivery;Persist Security Info=True;User ID=DigisticsStoreDeliveryUser;Password=password</DatabaseConnectionString>
  <DatabaseName>Name of the target database.</DatabaseName>
  <DatabaseUserName>User name of the user that has permission to read and write to the database.</DatabaseUserName>
  <DatabaseUserPassword>Password for the database user.</DatabaseUserPassword>
  <SaveOrmAssembly>false</SaveOrmAssembly>
  <HostAddressSuffix>Figlut</HostAddressSuffix>
  <PortNumber>8889</PortNumber>
  <TextResponseEncoding>UTF8</TextResponseEncoding><!--UTF8/ASCII/BigEndianUnicode/Default/Unicode/UTF32/UTF7-->
  <IncludeOrmTypeNamesInJsonResponse>true</IncludeOrmTypeNamesInJsonResponse>
</FiglutServerSettings>
