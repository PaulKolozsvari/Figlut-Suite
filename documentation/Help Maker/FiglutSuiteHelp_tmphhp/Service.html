The following settings can be configured under the Service category:

Ø   Host Address Suffix : The suffix to append to the URI on which the Figlut Server will be accessed i.e. http://localhost:{port_number}/{suffix} e.g. http://localhost:8889/Figlut.
Ø   Port Number : The port number on which the Figlut Server should listen for requests from clients i.e. http://localhost:{port_number}/{suffix} e.g. http://localhost:8889/Figlut.
Ø   Text Response Encoding : Encoding to used on the text response from the service.
Ø   Include ORM Type Names in JSON Response : Whether or not to include in the JSON response the names of the .NET generated ORM (Object Relational Mapping) types representing each table in the database.
