The settings for the Figlut Web Service may also be edited via the Figlut Server Manager, which is an MMC (Microsoft Management Console) snap-in. The Figlut Server Manager can accessed in one of two ways:


1.   From the Windows Start menu
1.   Start -> Programs -> Figlut -> Figlut Server Manager
2.   Running MMC and adding the Figlut Server Manager Snap-in:
1.   From the Windows Run Dialog Box, type "mmc" and press enter, which should open up the Microsoft Management Console (MMC)
2.   Click File -> Add/Remove Snap-in...
3.   Select the Figlut Server Manager snap-in and click Add. Then click OK.

Under the Figlut Web Service, there are following categories of settings enabling you to configure the behaviour of the Figlut Web Service:

Ø   Database
Ø   Service
Ø   Logging
