The following settings can be configured under the Database category:

Ø   DBMS : Database Management System being utilized (MS SQL Server is currently the only supported DBMS).
Ø   Connection String : The connection string to the target database (this setting overrides the below database credentials).
Ø   Name : Name of the target database.
Ø   User Name : User name of the user that has permission to read and write to the database.
Ø   Password : Password of the user accessing the database.
Ø   Save ORM Assembly : Whether or not to save generated assembly (.dll) that contains the ORM classes mapped to the database tables.
