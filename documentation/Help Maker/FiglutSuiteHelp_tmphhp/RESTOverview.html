A RESTful web service is an HTTP based web service implementing the principles of REST (Representational State Transfer). It represents a collection of resources which can be accessed through a base URI such as http://www.mydomainname.com/resources. The RESTful web service will also support various Internet media types such as XML or JSON and supports a set of operations using HTTP methods i.e. POST, PUT, DELETE or GET.
